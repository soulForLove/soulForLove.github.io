Nginx 禁止IP访问

为了避免别人把未备案的域名解析到自己的服务器IP而导致服务器被断网

nginx 出现413 Request Entity Too Large问题的解决方法

nginx默认上传文件的大小是1M，可nginx的设置中修改

  1. 打开nginx配置文件 nginx.conf

  2. 在http{}段中加入 client_max_body_size 20m; 20m为允许最大上传的大小

  3. 保存后重启nginx
